Help

At this page you can get help if you have problems with the system.

Logging into the system
To be able to log into the system you need to have a valid KTH account. If you have an account and you still can not log in, the problem is not in the KTH-EXIT system. If that is the case, you need to contact KTH administration.

Frequently asked questions (FAQ)
How do i log in to EXIT
You log in to the system with your KTH account. If you are new to the system you need to be added first.
The easiest way to log in is through the "Log in menu" via www.Exit.se
